CSVフォーマット

最大4ファイルに分けてインポートできます。nodes.csv と edges.csv は必須です。

nodes.csv

id,name,node_type,owner,update_frequency,description,tags
tbl_orders,受注テーブル,table,営業チーム,リアルタイム,受注情報,"sales,transactional"
etl_daily,日次ETL,etl,基盤チーム,日次,CRM→DWH同期,etl
sys_crm,CRMシステム,system,営業チーム,リアルタイム,Salesforce,"crm,external"

edges.csv

source_id,target_id,label,description
sys_crm,etl_daily,抽出,CRMからデータ抽出
etl_daily,tbl_orders,ロード,DWHへロード

columns.csv（テーブル/ファイルのスキーマ情報）

node_id,column_name,data_type,description,is_primary_key
tbl_orders,order_id,STRING,受注ID,true
tbl_orders,amount,INT64,金額,false

scripts.csv（ETLのスクリプト内容）

node_id,language,content
etl_daily,sql,"SELECT * FROM orders WHERE updated_at > ..."

node_type: table / file / system / etl
tagsはダブルクォート内にカンマ区切り。idは省略可（自動採番）
