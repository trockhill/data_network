JSONフォーマット

以下の形式でJSONファイルを作成してください。
nodes と edges の2つのキーが必要です。

{
  "nodes": [
    {
      "id": "tbl_orders",
      "name": "受注テーブル",
      "node_type": "table",
      "owner": "営業チーム",
      "update_frequency": "リアルタイム",
      "description": "受注ヘッダー情報",
      "tags": ["sales", "transactional"]
    }
  ],
  "edges": [
    {
      "id": "edge_001",
      "source_id": "sys_crm",
      "target_id": "scr_etl_daily",
      "label": "抽出",
      "description": "CRMからデータを抽出"
    }
  ]
}

CSVフォーマット

nodes.csv と edges.csv の2ファイルが必要です。

【nodes.csv】
id,name,node_type,owner,update_frequency,description,tags
tbl_orders,受注テーブル,table,営業チーム,リアルタイム,受注情報,"sales,transactional"
sys_crm,CRMシステム,system,営業チーム,リアルタイム,Salesforce,"crm,external"

【edges.csv】
id,source_id,target_id,label,description
edge_001,sys_crm,scr_etl_daily,抽出,CRMからデータを抽出

node_type: table / file / system / script
tagsはダブルクォート内にカンマ区切りで記述
